Regresssion Model

Now, we will run the following regression model to quantify the impact of Unrate on the GNP.

GNP=α+β×Unrate+ϵ

where, α is an intercetp, which defines the value of GNP when Unrate is zero. β value quantify the impact of change in GNP when Unrate changes and ϵ is an error term.

21.5 μs

We will use GML.jl package for estimating our regression model.
